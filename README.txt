运行Matlab 2019b以上的版本，打开app.mlapp文件，这时候会跳转到Matlab App Designer工具中，点击上方工具栏的“运行”，即可运行程序。

FSK传输数据：
在“收发数据”标签页中，一方先点击“接受”开始录音，另一方在输入框中填入文本，点击“发送”，发送完毕后接收方再次点击“接收”，即可结束录音，等待结果展示在结果框中。

Beepbeep测距：
在“声波测距”标签页中。
两台设备需要连接在同一局域网内，设备分为设备A和设备B。设备A填写自己的开放的端口，填写dA、dB距离修正参数，然后点击“测距A”。然后设备B填写设备A的局域网IP地址、端口号，点击“测距B”。即可开始测距。等待beepbeep结束后，测距结果会显示在设备A的图形界面中。
